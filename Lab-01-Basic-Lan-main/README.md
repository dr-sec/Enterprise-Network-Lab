# Cisco Packet Tracer - Basic LAN

## Objective

Create a small LAN consisting of two PCs connected to a Cisco switch and verify connectivity.

## Skills Learned

- IP Addressing
- Switching
- Basic Connectivity Testing
- ICMP (Ping)

## Network Diagram

PC1 ----- Switch ----- PC2

## Configuration

PC1
192.168.1.10

PC2
192.168.1.20

Subnet Mask
255.255.255.0

## Validation

Successfully pinged both devices.

## What I Learned

- Devices must be on the same subnet to communicate.
- Switches forward frames using MAC addresses.
- Ping uses ICMP to verify connectivity.
